# HWTestAgent Frontend

## 개요

HWTestAgent의 웹 대시보드입니다. Next.js 15 기반으로 구축되었으며, 에러 패턴, 테스트 결과, 템플릿을 관리할 수 있습니다.

## 주요 기능

### 1. 대시보드 홈 (`/`)
- 테스트 성공률 요약
- 최근 테스트 결과 (최신 5개)
- 주요 에러 패턴 (발생 빈도 높은 5개)

### 2. 에러 패턴 목록 (`/error-patterns`)
- 전체 에러 패턴 조회
- 필터링: 프로젝트, 카테고리, 검색

### 3. 에러 패턴 상세 (`/error-patterns/[id]`)
- 에러 패턴 상세 정보
- 관련 솔루션 목록

### 4. 테스트 결과 목록 (`/test-results`)
- 최근 50개 테스트 실행 결과
- 필터링: 프로젝트, 상태

### 5. 템플릿 목록 (`/templates`)
- 재사용 가능한 테스트 스크립트 템플릿
